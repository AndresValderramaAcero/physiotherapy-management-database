## Overview

This repository contains the SQL Server database schema and stored procedures for a physiotherapy patient management system developed during my professional internship.

The project includes the relational database design and the stored procedures used by a C# application to manage patient evaluations and clinical records.

> Note: Any institution-specific information and sensitive data have been removed to preserve confidentiality.

## Features

- Patient management
- Clinical evaluation records
- Relational database design
- Stored procedures
- SQL Server

## Technologies

- Stored Procedures
